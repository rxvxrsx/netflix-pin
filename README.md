# Netflix PIN Auto-Fill

Chrome Extension สำหรับช่วยกรอก PIN บนหน้า Netflix ผ่าน Chrome Side Panel โดยสามารถตั้งค่าเลขเริ่มต้น ลำดับการลอง PIN และความเร็วในการกรอกได้จากแผงควบคุมเดียว

> ใช้กับบัญชีและอุปกรณ์ที่คุณมีสิทธิ์ใช้งานเท่านั้น โปรเจกต์นี้ทำขึ้นเพื่อช่วย automation ส่วนตัว ไม่ควรใช้กับบัญชีหรือข้อมูลของผู้อื่นโดยไม่ได้รับอนุญาต

## ฟีเจอร์หลัก

- ตั้งค่า PIN เริ่มต้นได้ตั้งแต่ `0000` ถึง `9999`
- เลือกลำดับการลอง PIN ได้ 2 แบบ:
  - `น้อยไปมาก` เช่น `0000` ไป `9999`
  - `มากไปน้อย` เช่น `9999` ไป `0000`
- ปรับเวลาระหว่างการกดแต่ละปุ่มได้
- ปรับเวลาระหว่างการลองแต่ละรหัสได้
- แสดง PIN ล่าสุด สถานะการทำงาน และบันทึก log
- ส่งคำสั่งไปยังแท็บ Netflix ที่เปิดอยู่เท่านั้น
- ใช้งานผ่าน Chrome Side Panel ไม่ต้องเปิดหน้าต่างแยก

## วิธีติดตั้ง

1. เปิด Chrome แล้วไปที่ `chrome://extensions`
2. เปิด `Developer mode`
3. กด `Load unpacked`
4. เลือกโฟลเดอร์โปรเจกต์นี้
5. เปิดหน้า Netflix
6. กดไอคอน extension เพื่อเปิด Side Panel

## วิธีใช้งาน

1. กรอกค่า `เริ่มต้นจาก PIN`
2. เลือก `ลำดับการลอง PIN`
3. ตั้งค่า delay ตามต้องการ:
   - `เวลาระหว่างการกดแต่ละปุ่ม` คือเวลาหน่วงระหว่างเลขแต่ละหลัก
   - `เวลาระหว่างแต่ละรหัส` คือเวลาหน่วงก่อนลอง PIN ถัดไป
4. กด `บันทึกค่า` ถ้าต้องการเก็บ config ไว้ใช้ครั้งต่อไป
5. กด `เริ่มทำงาน`
6. กด `หยุดทำงาน` เมื่อต้องการหยุด

## ตัวอย่างการตั้งค่า

| ต้องการ | PIN เริ่มต้น | ลำดับการลอง |
| --- | --- | --- |
| ลองจากเลขน้อยไปหาเลขมาก | `0000` | น้อยไปมาก |
| ลองจากเลขมากลงมาเลขน้อย | `9999` | มากไปน้อย |
| เริ่มต่อจากเลขกลางช่วง | `4500` | น้อยไปมาก |
| ถอยลงจากเลขกลางช่วง | `4500` | มากไปน้อย |

## โครงสร้างไฟล์

| ไฟล์ | หน้าที่ |
| --- | --- |
| `manifest.json` | กำหนดค่า Chrome Extension |
| `background.js` | เปิด Side Panel และส่งคำสั่งไปยังแท็บ Netflix |
| `content.js` | ทำงานบนหน้า Netflix เพื่อกรอก PIN |
| `popup.html` | หน้าตา Side Panel |
| `popup.js` | จัดการ UI, config, สถานะ และ log |

## การตั้งค่าที่แนะนำ

- เริ่มจาก delay ปานกลางก่อน เช่น `keyDelay = 80` และ `codeDelay = 120`
- ถ้าหน้าเว็บตอบสนองช้า ให้เพิ่ม `codeDelay`
- ถ้าเลขกรอกไม่ครบหรือข้ามจังหวะ ให้เพิ่ม `keyDelay`
- Reload extension หลังแก้ไฟล์ทุกครั้งที่ `chrome://extensions`

## หมายเหตุ

- Extension นี้ไม่ได้แก้ไขระบบของ Netflix และทำงานเฉพาะใน browser ของผู้ใช้
- หน้าเว็บ Netflix อาจเปลี่ยนโครงสร้างได้ ทำให้ selector บางตัวต้องปรับตามในอนาคต
- ควรใช้อย่างระมัดระวังและอยู่ภายใต้สิทธิ์การใช้งานของบัญชีตนเอง
