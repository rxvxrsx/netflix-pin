# Netflix PIN Auto-Fill

Chrome extension สำหรับช่วยกรอก PIN บนหน้า Netflix ผ่าน Chrome Side Panel และมีเครื่องมือเสริมสำหรับเปิดหน้า `Browse` กับดึงข้อมูลแพ็กเกจจาก `https://www.netflix.com/account`

> ใช้กับบัญชีและอุปกรณ์ที่คุณมีสิทธิ์ใช้งานเท่านั้น โปรเจกต์นี้ทำเพื่อช่วย automation ส่วนตัว ไม่ใช่การเข้าถึงข้อมูลของผู้อื่น

## ฟีเจอร์

- ตั้งค่า PIN เริ่มต้นได้ตั้งแต่ `0000` ถึง `9999`
- เลือกลำดับการลอง PIN ได้ 2 แบบ
  - น้อยไปมาก เช่น `0000` -> `9999`
  - มากไปน้อย เช่น `9999` -> `0000`
- ปรับ delay ระหว่างกดปุ่มและระหว่างรหัสได้
- แสดง PIN ล่าสุด สถานะการทำงาน และบันทึก log
- ดึงข้อมูลแพ็กเกจ Netflix จากหน้า `account` อัตโนมัติ
- แสดงชื่อแพ็กเกจพร้อมแยกสีตามระดับ เช่น `Mobile`, `Basic`, `Standard`, `Premium`
- ปุ่มกลับไปหน้า `https://www.netflix.com/browse`
- ใช้งานผ่าน Chrome Side Panel ไม่ต้องเปิดหน้าต่างแยก

## วิธีติดตั้ง

1. เปิด Chrome แล้วไปที่ `chrome://extensions`
2. เปิด `Developer mode`
3. กด `Load unpacked`
4. เลือกโฟลเดอร์โปรเจกต์นี้
5. เปิดหน้า Netflix ที่ต้องการใช้งาน
6. กดไอคอน extension เพื่อเปิด Side Panel

## วิธีใช้งาน

1. กรอกค่า `เริ่มต้นจาก PIN`
2. เลือก `ลำดับการลอง PIN`
3. ปรับค่า delay ตามต้องการ
4. กด `บันทึกค่า` หากต้องการเก็บ config ไว้
5. กด `เริ่มทำงาน` เพื่อเริ่มกรอก PIN
6. กด `หยุดทำงาน` เมื่อต้องการหยุด
7. กด `ดึงข้อมูลแพ็กเกจ` เพื่อให้ extension เปิดหน้า `https://www.netflix.com/account` และอ่านชื่อแพ็กเกจ
8. กด `กลับหน้า Browse` เพื่อพาแท็บ Netflix ไปที่ `https://www.netflix.com/browse`

## ตัวอย่างค่าที่ตั้ง

| เป้าหมาย | PIN เริ่มต้น | ลำดับการลอง |
| --- | --- | --- |
| ลองจากเลขน้อยไปหาเลขมาก | `0000` | น้อยไปมาก |
| ลองจากเลขมากลงมา | `9999` | มากไปน้อย |
| เริ่มจากเลขกลางช่วง | `4500` | น้อยไปมาก |

## โครงสร้างไฟล์

| ไฟล์ | หน้าที่ |
| --- | --- |
| `manifest.json` | กำหนดค่า Chrome extension |
| `background.js` | รับคำสั่งจาก popup และจัดการแท็บ Netflix |
| `content.js` | ทำงานบนหน้า Netflix เพื่อกรอก PIN และอ่านข้อมูลแพ็กเกจ |
| `popup.html` | หน้าตา Side Panel |
| `popup.js` | จัดการ UI, config, status, log และปุ่มเสริม |

## หมายเหตุ

- Extension นี้ทำงานเฉพาะใน browser ของผู้ใช้
- หน้า Netflix อาจเปลี่ยนโครงสร้าง DOM ได้ ทำให้ selector บางส่วนต้องปรับตามอนาคต
- ควรใช้ด้วยความระมัดระวังและอยู่ภายใต้สิทธิ์การใช้งานของบัญชีตัวเอง
